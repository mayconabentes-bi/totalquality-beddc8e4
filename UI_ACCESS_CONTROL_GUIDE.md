## Color Scheme

- **Primary Action Button**: Blue/Primary color
- **Secondary Button**: Outline/Gray
- **Success State**: Green (#10b981)
- **Disabled State**: Gray (#9ca3af)
- **Selected Item**: Light blue background
- **Role Badges**:
  - Master: Purple (#9333ea)
  - Proprietario: Blue (#2563eb)
  - Secretaria: Green (#16a34a)
  - Treinador: Orange (#ea580c)

## Interactive States

### Switch Component
- OFF: Gray circle on left, light gray background
- ON: Primary color circle on right, primary background

### Checkbox
- Unchecked: Empty square with border
- Checked: Blue square with white checkmark

### Button States
- Default: Colored background
- Hover: Slightly darker
- Disabled: Gray with reduced opacity
- Active/Pressed: Even darker

## Responsive Behavior

### Desktop (>768px)
- Modal: max-width 768px, centered
- Settings: Full width with padding
- Dashboard: 2-4 column grid

### Mobile (<768px)
- Modal: Full width, scrollable
- Settings: Single column
- Dashboard: Single column stack
